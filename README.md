# Expense Tracker Pro 💰

A simple and efficient Expense Tracker built with Python.

## Features

- Add expenses
- View expense history
- Calculate total expenses
- Category-wise spending summary
- CSV file storage
- User-friendly terminal interface

## Project Structure

```bash
expense-tracker-pro/
│
├── main.py
├── expenses.csv
├── README.md
├── requirements.txt
└── reports/
```

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/expense-tracker-pro.git
```

Navigate to the project:

```bash
cd expense-tracker-pro
```

Run the application:

```bash
python main.py
```

## Usage

### Add Expense

Enter:
- Category
- Description
- Amount

Example:

```text
Category: Food
Description: Lunch
Amount: 250
```

### View Expenses

Displays all saved expenses.

### Category Summary

Shows total spending per category.

Example:

```text
Food: ₹2500
Travel: ₹1500
Shopping: ₹3000
```
